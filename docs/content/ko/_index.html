Goployer는 AWS 오토스케일링, 로드 밸런싱을 활용하여 전체 배포 프로세스를 관리합니다.
{{% blocks/lead color="primary" %}} Goployer는 AWS 오토스케일링을 활용한 인스턴스 배포 전체 프로세스를 통제합니다. Goployer는 새로운 버전의 오토스케일링 그룹을 생성하고, 먼저 로드밸런서의 타겟그룹에 붙입니다. 해당 인스턴스들이 전부 정상임이 확인되면, 그 때 이전 버전의 오토스케일링 그룹들을 삭제합니다. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="프로덕션 수준" %}} Goployer는 오토스케일링을 활용한 배포에 있어서 대부분의 기능을 제공합니다.
